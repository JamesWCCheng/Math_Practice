一元一次方程式 — 第一單元 練習卷 (Set 4)

共 20 題｜每題解出未知數的值｜出處標示於題號右側

🔵 第一組：分數與小數混合（H21–H25 不同題）
1. $\dfrac{x}{3} + \dfrac{x}{4} = \dfrac{7}{12}$ 出處：H21a, 第15題
2. $\dfrac{x}{6} - \dfrac{1}{4} = \dfrac{x}{4} - \dfrac{1}{3}$ 出處：H21b, 第18題
3. $\dfrac{x}{5} + \dfrac{x}{3} = \dfrac{8}{15}$ 出處：H22a, 第14題
4. $\dfrac{2x+1}{5} - \dfrac{x-2}{3} = \dfrac{1}{15}$ 出處：H22b, 第16題
5. $\dfrac{2}{3}x - 4 = -\dfrac{4}{5}x + 6$ 出處：H23a, 第13題
6. $\dfrac{7}{10}x - \dfrac{3}{5} = \dfrac{1}{2}x + \dfrac{2}{5}$ 出處：H23b, 第15題
7. $3.6x - 2.4 = 1.2x + 4.8$ 出處：H24a, 第16題
8. $\dfrac{9}{10}x - \dfrac{3}{4} = \dfrac{2}{5}x + \dfrac{1}{4}$ 出處：H24b, 第18題
9. $2x - \dfrac{4x + 3}{5} = \dfrac{x - 2}{3}$ 出處：H25a, 第16題
10. $\dfrac{3}{4}x - \dfrac{1}{6} = \dfrac{5}{12}x + \dfrac{1}{3}$ 出處：H25b, 第17題
🟣 第二組：高階多層括號（H46–H50 不同題）
11. $\dfrac{2x}{3} + \dfrac{3}{25} = \dfrac{1}{5} + \dfrac{x}{5}$ 出處：H46a, 第12題
12. $0.5(0.3x - 2) = 0.2(0.6x + 3)$ 出處：H46b, 第17題
13. $2y - \dfrac{7y - 3}{9} = -6$ （未知數為 $y$） 出處：H47a, 第14題
14. $\dfrac{7}{4}(x - 2) + 0.25x = \dfrac{4x + 1}{8}$ 出處：H47b, 第16題
15. $4x - \dfrac{9x + 13}{5} = -\dfrac{13 + 3x}{7}$ 出處：H48a, 第10題
16. $\dfrac{1}{5}(x - 2) = \dfrac{7}{6} - \dfrac{1}{3}(4x + 1)$ 出處：H48b, 第14題
17. $\dfrac{x - 3}{5} - \dfrac{2x + 1}{4} + \dfrac{3x - 7}{10} = 1$ 出處：H49a, 第11題
18. $2\{x - 4(x - 2)\} = 5(x + 1) - 3$ 出處：H49b, 第15題
19. $\dfrac{3 - 4x}{5} + 3 = 2 - \dfrac{4x - 9}{7} - 3x$ 出處：H50a, 第8題
20. $\dfrac{2x - 1}{3} - \dfrac{4x + 3}{9} - \dfrac{2}{3} = \dfrac{5x - 4}{7} - \dfrac{4x - 1}{21}$ 出處：H50b, 第14題
