第四單元：一元一次方程應用題 — 練習卷 (Set 1)

共 15 題｜每題求聯立解｜出處標示於題號右側

1. x 的3倍比 x 多10。 出處：H142
2. x 除以3後比 x 小8。 出處：H142
3. x 減5後所得數的2倍等於6。 出處：H142
4. x 加2所得數的 3/4 等於9。 出處：H142
5. x 加13.5等於28。 出處：H143
6. x 的3倍加5等於26。 出處：H143
7. x 的2倍加16所得的數等於85減 x 所得的數。 出處：H143
8. x 的 2/3 減5，等於 x 除以4。 出處：H143
9. 大華原有的錢加200元後再擴大3倍，等於原有錢數的8倍，問大華原有多少錢？ 出處：H146
10. A有400元，B有1100元。A買了2張畫、B買了7張畫後，兩人所剩的錢相等。問一張畫多少錢？ 出處：H146
11. 張家有存款1000元，李家有存款100元。從現在開始張家每月存儲300元，李家每月存儲200元。問幾個月後張家的存款等於李家存款的2倍？ 出處：H146
12. 原有的錢買5台收音機剩50元，而買8台收音機還差100元，問每台收音機多少錢？ 出處：H147
13. 把桔子分給參加生日聚會的孩子們，每人分6個則剩下23個，每人分9個則剩下5個。問參加生日聚會的孩子有幾人？ 出處：H147
14. 把桔子分給全家人，每人分5個則剩下15個，每人分8個則差3個。問全家有幾口人？ 出處：H147
15. 把1箱桔子平均分給5人比平均分給8人每人可多得3個。問一箱有多少個桔子？ 出處：H148
