📙 第一組：括號與分配律（H26–H30）
1. $5(2x - 3) = 3(x + 4)$ 出處：H26a, 第10題
2. $3(x - 2) - 4(2x + 1) = -5x + 8$ 出處：H26b, 第8題
3. $5(x + 3) - 3(2x - 1) = 4(x - 2)$ 出處：H27a, 第12題
4. $4(3x - 2) - 5(2x + 3) = 2(x - 7)$ 出處：H27b, 第5題
5. $2(x + 4) - (3x - 5) = -x + 13$ 出處：H28a, 第10題
6. $3x + 2(x - 4) = \dfrac{1}{2}(x + 6)$ 出處：H28b, 第16題
7. $6(3x - 1) + 2 = 4(5x + 2) - 10$ 出處：H29a, 第13題
8. $5 - 3(4x - 2) = 2 - (3x + 5)$ 出處：H29b, 第7題
9. $-\dfrac{1}{4}x + \dfrac{5}{8} = \dfrac{3}{8}x - \dfrac{3}{4}$ 出處：H30a, 第11題
10. $0.4(3.5x - 2.5) = 0.6(2x - \dfrac{5}{3})$ 出處：H30b, 第8題
📕 第二組：高階綜合挑戰（H46–H50）

15. $3x - \dfrac{7x + 11}{4} = -\dfrac{11 + x}{6}$ 出處：H48a, 第7題
16. $\dfrac{1}{4}(x - 3) = \dfrac{5}{6} - \dfrac{1}{3}(2x + 1)$ 出處：H48b, 第11題

20. $\dfrac{x - 1}{5} - \dfrac{3x - 2}{8} - \dfrac{1}{4} = \dfrac{2x - 3}{6} - \dfrac{3x + 4}{12}$ 出處：H50b, 第10題
